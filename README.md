# express版 api server

[![Greenkeeper badge](https://badges.greenkeeper.io/zanjs/Julian-api-express.svg)](https://greenkeeper.io/)



## 环境准备
```
node.js 4.0+
mognodb 3.0+
```

## 开发
```
$ npm install
$ npm start
```
配合客户端测试的测试模式   
 
```
$ gulp serve:test
```


## 测试
```
$ npm run test
```
